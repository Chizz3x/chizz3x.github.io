# Chizz3x.github.io

A portfolio project that is focused to introduce unique and interactive portfolio both on desktop and mobile browsers.

## Idea

To have a Workdesk UI much like Windows or Linux and create custom apps that present projects I work or worked on.

Main view is always the Workdesk that allows all basic functionality of moving the apps, opening them, if its a window, reisizing and such.

## Base app

The existing base app is created to present "about me" in a console way. This si achieved by rendering everything in canvas for performance.

## Other apps

Other apps will be added and they will either link to other places or work in the same portfolio site.

## User features

- Sould be able to move and swap apps.
- Should be able to resize the window screen live and icon should adjust optimally in the grid.
- Apps should be openable.
- Whoel project has to be optimal and fast.
- Work both on mobile and desktop environments.
- Allow creating multiple desktops via shortcuts or extra navigation toolbar for mobile.
- Extra desktops are created based on user command and on demand by the initial ssytem start on client if apps overflow.
